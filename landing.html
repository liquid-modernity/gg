Structure

The page has jobs, not ornaments.

HTML carries meaning and hooks. CSS carries visual rhythm, layout, motion, responsive rules, and state styling. JS only orchestrates behavior, copy binding, state toggles, gestures, and events.

HTML/body

Semantic sections, heading order, real links, ARIA, data hooks, and machine-readable summary stay in the document.

CSS/style

The visual system inherits the GG dock, outline, sheet, token, contrast, reduced-motion, and responsive contracts.

JS/script

JavaScript upgrades the surface with panels, gestures, keyboard handling, language, theme, scroll state, and discovery.

No-JS truth

Important content remains readable without script. Interaction is enhancement, not the source of truth.

Interaction

The dock is primary. The outline is contextual.

The bottom dock keeps route navigation stable. The contents outline attaches above it, then drops to the bottom when the dock hides on scroll. Tap, swipe, keyboard focus, and mouse hover all have a defined state.

  • Touch: tap the outline to expand, swipe up to open, swipe down to collapse.
  • Keyboard: Tab reaches the dock, outline, sheet controls, and route links; Escape closes open panels.
  • Mouse: hover is subtle and never required; focus states are visible.
  • Motion: section handoff explains state, but reduced motion disables non-essential animation.
