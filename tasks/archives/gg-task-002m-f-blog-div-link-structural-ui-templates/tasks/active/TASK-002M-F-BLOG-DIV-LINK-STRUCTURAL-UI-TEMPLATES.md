# TASK-002M-F — Migrate Blog div/link Structural UI to Templates

Ready for implementation by AI code agent.

## Acceptance criteria

- `apps/blog/index.xml` contains purpose-specific templates for the migrated UI structures.
- `src/modules/legacy-app/legacy-app.js` uses template cloning for these UI structures.
- JS only fills state/aria/text/href/data/icon after clone.
- No universal/generic template is introduced.
- `npm run check:public-dom` passes with `unclassified=0` and no remaining known `needsTemplate` candidates.
- Full pipeline passes:
